Com gestionar l'arbre

1 Quan arribi una proposta per email

Formspree t'enviarà un email a amb totes les dades de la persona proposada. El flux és:

  1. Reps l'email de Formspree amb les dades
  2. Si ho aprovaves: afegeix la persona al familia.json (veure bloc 2)
  3. Puges el fitxer actualitzat a GitHub → l'arbre s'actualitza automàticament
  4. Si ho rebutges: simplement esborra l'email

2 Afegir una persona al JSON

Cada persona al familia.json té aquesta forma. Copia, omple i enganxa al fitxer:

{
  "id": "p21",
  "nom": "Nom",
  "cognoms": "Cognom1 Cognom2",
  "genere": "home",
  "data_naix": "1950-03-15",
  "lloc_naix": "Barcelona",
  "data_mort": null,
  "lloc_mort": null,
  "foto_url": null,
  "notes": null,
  "pare_id": "p01",
  "mare_id": "p02"
}

• id: segueix la seqüència (p21, p22…) · • genere: "home", "dona" o "altre" · • data_naix: format AAAA-MM-DD · • pare_id/mare_id: l'id de la persona pare/mare, o null

Si la persona té parella, afegeix també una línia a "unions":
{ "id": "u08", "persona1_id": "p21", "persona2_id": "p14" }

✓ Copiat!

3 Pujar els canvis a GitHub

Opció A — Des del navegador (més fàcil):

  1. Ves al teu repositori a github.com
  2. Fes clic a familia.json
  3. Fes clic al llapis ✏️ (Edit this file)
  4. Afegeix la nova persona al JSON
  5. A baix: "Commit changes" → guarda

Opció B — Des del terminal (si tens Git):

git add familia.json
git commit -m "Afegeix [nom] a l'arbre"
git push

⚡ GitHub Pages actualitza l'arbre en menys d'un minut.

4 Afegir una foto

  1. Puja la foto a qualsevol servei gratuït: Google Photos, Dropbox, ImgBB…
  2. Obtén l'URL directa de la imatge (ha d'acabar en .jpg o .png)
  3. Posa l'URL al camp "foto_url" de la persona al JSON

Per ImgBB (recomanat): ves a imgbb.com, puja la foto, copia el Direct link.
